# space-invaders

# How to Play

Use Left Arrow and Right Arrow to move the player.
Press Space to shoot lasers.
Destroy all aliens before they reach the bottom.
Avoid enemy lasers.
You have 3 lives.

# How to run
open CMD

#clone:
```
git clone "https://github.com/fabiana-leticia/space-invaders.git"
```
#go to project folder:
```
cd space-invaders
```
#install the required packeges:
```
pip install -r requirements.txt
```
#Run the main.py file:
```
python main.py
```
